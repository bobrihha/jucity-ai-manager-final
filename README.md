# Джунгли Сити AI Bot

Telegram бот для сети парков "Джунгли Сити" с AI-ассистентом.

## Установка

```bash
pip install -r requirements.txt
```

## Настройка

1. Скопируйте `.env.example` в `.env`
2. Заполните переменные окружения

## Запуск

```bash
# Бот
python -m bot.main

# Админка
streamlit run admin/app.py
```
